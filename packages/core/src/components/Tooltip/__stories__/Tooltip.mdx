# Tooltip

Tooltips provide helpful information and can include detailed context about a specific component when a user hovers on it.

<Canvas of={TooltipStories.Overview} />

### Import

```js
import { Tooltip } from "@vibe/core";
```

## Variants

### Tooltip with title

Use tooltip with title when you want to present more complex content and you need to add a heading to emphasize the main idea. Titles can also come with an icon.

<Canvas of={TooltipStories.TooltipWithTitle} />

### Tooltip with image

Use tooltip with image to provide a preview for an image or when giving a closer look at a feature.

<Canvas of={TooltipStories.TooltipWithImage} />

### Positions

Tooltip's arrow can appear from top, bottom, left or right.

## Do's and Don'ts

<ComponentRules
  rules={[
    {
      positive: {
        component: (
          <Tooltip shouldShowOnMount content="Quick search + B" modifiers={modifiers}>
            <IconButton kind="secondary" icon={Bolt} iconSize="32" />
          </Tooltip>
        ),
        description: "Use tooltips for interactive imagery."
      },
      negative: {
        component: (
          <Tooltip shouldShowOnMount content="Quick search + B" modifiers={modifiers}>
            <Button>Quick search</Button>
          </Tooltip>
        ),
        description: "Don't use tooltips to restate visible UI text."
      }
    },
    {
      positive: {
        component: (
          <TabList>
            <Tab>Boards</Tab>
            <Tab>Items</Tab>
          </TabList>
        ),
        description: "If an element doesn't need additional clarification, don't use a tooltip."
      },
      negative: {
        component: (
          <TabList>
            <Tooltip shouldShowOnMount content="Boards" position="bottom" modifiers={modifiers}>
              <Tab>Boards</Tab>
            </Tooltip>
            <Tab>Items</Tab>
          </TabList>
        ),
        description: "Don't use tooltips to restate text that is already displayed within the element."
      }
    },
    {
      positive: {
        component: (
          <Tooltip
            shouldShowOnMount
            title="I'm a rich tooltip"
            modifiers={modifiers}
            open
            content={
              <div>
                I have a title, text, image and I can come <Link text="with a link" inlineText color="onPrimary" />.
              </div>
            }
          >
            <div style={{ position: "relative", top: "25%" }} />
          </Tooltip>
        ),
        description: "Add links inline and in the same color of the text."
      },
      negative: {
        component: (
          <Tooltip
            shouldShowOnMount
            title="I'm a rich tooltip"
            modifiers={modifiers}
            open
            content={
              <div>
                I have a title, text, image and I can come with a link. <Link text="Read more" />
              </div>
            }
          >
            <div style={{ position: "relative", top: "25%" }} />
          </Tooltip>
        ),
        description: "Don't add links on a separate line or in a different color from the text."
      }
    },
    {
      positive: {
        component: (
          <Tooltip
            content="Empower users to visualize data."
            shouldShowOnMount
            image={image}
            position="right"
            modifiers={modifiers}
            open
          >
            <div style={{ position: "relative", left: "-25%" }} />
          </Tooltip>
        ),
        description: "Use this app to visualize data."
      },
      negative: {
        component: (
          <Tooltip
            content="View our automation center."
            shouldShowOnMount
            image={image}
            position="right"
            modifiers={modifiers}
            open
          >
            <div style={{ position: "relative", left: "-25%" }} />
          </Tooltip>
        ),
        description: "Don't add an image just for aesthetic reasons, or if it doesn't help the user."
      }
    }
  ]}

## Use cases and examples

### Icon tooltip

An icon tooltip is used to clarify the action or name of an interactive icon button. Provide tooltips for all unlabelled icons.

<Canvas of={TooltipStories.IconTooltip} />

### Definition tooltip

The definition tooltip provides additional help or defines an item or term. It may be used on the label of a UI element, or on a word embedded in a paragraph.

<Canvas of={TooltipStories.DefinitionTooltip} />

### Immediate tooltips

Immediately show when another is shown. The two left tooltips uses the immediate show prop, the right one ignores it and should always have show delay.

<Canvas of={TooltipStories.ImmediateTooltips} />
