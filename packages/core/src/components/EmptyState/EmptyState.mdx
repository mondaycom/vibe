# EmptyState

EmptyState is a generic, reusable UI component to communicate "no data" or "empty" scenarios. It centers optional decorative visuals, an optional title, a required description, and up to two actions (primary and secondary) to guide users toward the next steps.

## Import path

```tsx
import { EmptyState } from "@vibe/core"; 
```
