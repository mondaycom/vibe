# useEventListener

Attaches a listener to any DOM event on a specific element, firing a provided callback when the event is triggered.

<Canvas of={UseEventListenerStories.Overview} />

### Import

```js
import { useEventListener } from "@vibe/core";
```

## Arguments

<FunctionArguments>
  <FunctionArgument name="options" type="Object">
    <FunctionArgument
      name="ref"
      type="React.MutableRefObject"
      description={
        <>
          A React

          object.

    <FunctionArgument
      name="callback"
      type="(event: Event) => void"

    />
