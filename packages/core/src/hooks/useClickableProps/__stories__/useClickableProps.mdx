# useClickableProps

Return props for making an element or component clickable by mouse and keyboard with screen reader support.

<Canvas of={UseClickablePropsStories.Overview} />

### Import

```js
import { useClickableProps } from "@vibe/core";
```

<TipClickable />

## Usage

  guidelines={[
