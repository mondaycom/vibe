# Dropdown

Select component present a list of options from which a user can select one or several.

### Import

```js
import { Dropdown } from "@vibe/core";
```

### Basic dropdown

The <StorybookLink page="Components/Dropdown/Basic dropdown">Basic dropdown</StorybookLink> present a popup list of options from which a user can select one or several.

<Canvas of={BasicDropdownPreview} sourceState="none" />

### Dropdown box mode

The <StorybookLink page="Components/Dropdown/Dropdown box mode">Dropdown box mode</StorybookLink> is an always open Dropdown, with the same properties as the basic Dropdown.
