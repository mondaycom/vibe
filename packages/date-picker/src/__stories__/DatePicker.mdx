# DatePicker

A simple and reusable Datepicker component

<Canvas of={DatepickerStories.Overview} />

### Import

DatePicker is not included in the `@vibe/core` package. To use it, install and import from the separate `@vibe/date-picker` package:

```js
import { DatePicker } from "@vibe/date-picker";
```
